## Features

**IdeaMemo** 是一款 Android 轻量级以及标签 Tag 为核心的卡片便签App。

数据存储在您的手机，也可以通过 Webdav 上传到自己私有云存储，App使用不需要申请任何的权限。

以下是当前功能和未来计划：

- [x] 纯用 Compose 和 Kotlin 编写。
- [x] 支持 Markdown 书写。
- [x] 支持图片。
- [x] 支持 TAG 标签关联。
- [x] 支持主题和颜色。
- [x] 本地数据备份和恢复。
- [x] 以 JSON 格式导出。
- [x] Webdav 云数据备份和恢复。
- [x] 完成英文翻译。
